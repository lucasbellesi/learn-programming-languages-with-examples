# C++ Exercise Solution Rubric

Use this rubric to evaluate your exercise solutions before marking them complete.

## Scoring

Score each category from `0` to `2`:

- `0`: not met
- `1`: partially met
- `2`: fully met

Target: at least `7/8` before considering a solution done.

## Categories

### 1) Correctness

- Program solves the exact task in the exercise spec.
- Required output is produced for normal inputs.

### 2) Edge Cases

- Program handles the edge cases listed in module `README.md`.
- Invalid input is handled safely when required by the exercise.

### 3) Readability

- Variable/function names are clear and beginner-friendly.
- Control flow is straightforward and easy to follow.
- Output messages are understandable.

### 4) Compile Quality

- Compiles with:
  - `g++ -std=c++17 -Wall -Wextra -pedantic`
- No warnings introduced by the solution.
