# Operators and Expressions (Go)

## Quick Run

```bash
go run example/main.go
```

## Topics Covered

- Arithmetic operators: `+ - * / %`
- Comparison operators: `== != < <= > >=`
- Boolean expressions with `&&`, `||`, `!`
- Precedence in mixed expressions

## Exercise Focus

- `exercises/01.go`: convert total seconds to hours, minutes, and seconds.
- `exercises/02.go`: apply discount and tax in sequence.
