# Types and Input/Output (Go)

## Quick Run

```bash
go run example/main.go
```

## Topics Covered

- `int`, `float64`, `string`, `bool`
- `fmt.Print*` output
- `fmt.Scanln` and `fmt.Scanf` input

## Exercise Focus

- `exercises/01.go`: read N values and compute summary.
- `exercises/02.go`: parse product, price, quantity.
