# Arrays and Vectors (Go Slices)

## Quick Run

```bash
go run example/main.go
```

## Topics Covered

- Fixed arrays and dynamic slices
- Appending values with `append`
- Iterating with index loops and `range`
- Aggregations over collections

## Exercise Focus

- `exercises/01.go`: read values and print them in reverse order.
- `exercises/02.go`: count frequency of a target value.
