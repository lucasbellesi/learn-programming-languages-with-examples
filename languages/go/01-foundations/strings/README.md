# Strings (Go)

## Quick Run

```bash
go run example/main.go
```

## Topics Covered

- String normalization with `strings` package
- Character filtering
- Tokenization with `strings.Fields`
- Counting and comparing text

## Exercise Focus

- `exercises/01.go`: count words in a sentence.
- `exercises/02.go`: palindrome check ignoring punctuation and case.
