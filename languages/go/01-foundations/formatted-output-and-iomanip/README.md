# Formatted Output and iomanip Parity (Go)

## Quick Run

```bash
go run example/main.go
```

## Topics Covered

- Width and alignment with `fmt.Printf`
- Numeric precision formatting
- Tabular output for reports
- Reusable formatting patterns

## Exercise Focus

- `exercises/01.go`: print an invoice-like aligned table.
- `exercises/02.go`: print statistics with configurable precision.
