# 03 Advanced Capstone: Course Enrollment Manager

## Goal

Model courses and enrollments with structs, methods, and guarded state updates.

## Quick Run

```bash
go run main.go
```

## Requirements

- Define a `Course` struct with title and capacity.
- Provide methods for enroll, drop, and print status.
- Prevent enrollment beyond capacity.
- Demonstrate multiple course objects.

## Sample Output

```text
Course: GoBasics | 2/2 enrolled
Course: Algorithms | 1/3 enrolled
```

## Extension Ideas

- Add student IDs.
- Add waitlist behavior.
