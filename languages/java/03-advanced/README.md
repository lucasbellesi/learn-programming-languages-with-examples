# Java 03 Advanced

This level covers the full advanced Java module set with object modeling, constructor-enforced invariants, reference behavior, polymorphism, and generics.

## Learning Metadata

- Difficulty: Intermediate.
- Estimated Time: 4-5 hours.
- Prerequisites: Completion of `01-foundations` and `02-core`.
- Study Strategy: Focus on data plus behavior first, then compare Java classes, reference copying, interfaces, and generics with equivalent patterns in other tracks.

## Module Order

1. [structs-and-classes](./structs-and-classes/README.md)
2. [constructors-and-invariants](./constructors-and-invariants/README.md)
3. [copy-and-move-semantics](./copy-and-move-semantics/README.md)
4. [inheritance-and-polymorphism](./inheritance-and-polymorphism/README.md)
5. [templates-basics](./templates-basics/README.md)

## Level Outcomes

- Model related data and behavior with Java records and classes.
- Protect object state through constructors and methods.
- Define invariants that remain true after construction and updates.
- Explain reference assignment, aliasing, and defensive copies.
- Treat different concrete classes through one shared Java interface.
- Reuse logic with generic methods, generic classes, and simple type bounds.
- Compare Java object modeling with C++, C#, Go, Python, and TypeScript.
