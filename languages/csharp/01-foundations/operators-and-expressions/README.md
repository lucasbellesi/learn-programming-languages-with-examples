# Operators and Expressions (C#)

## Quick Run

```bash
dotnet run --project example/operators-and-expressions-example.csproj
```

## Topics Covered

- Arithmetic operators (`+ - * / %`)
- Comparison and boolean operators
- Expression precedence
- Integer vs floating-point operations

## Exercise Focus

- `exercises/01.cs`: convert total seconds to hours, minutes, and seconds.
- `exercises/02.cs`: apply discount and tax in sequence.
