# Scope and Lifetime Basics (C#)

## Quick Run

```bash
dotnet run --project example/scope-and-lifetime-basics-example.csproj
```

## Topics Covered

- Local variable scope
- Constants for shared configuration
- Variable lifetime in loops and blocks
- Reducing unnecessary mutable state

## Exercise Focus

- `exercises/01.cs`: assign grade with clean variable scope.
- `exercises/02.cs`: sum values with minimal variable lifetime.
