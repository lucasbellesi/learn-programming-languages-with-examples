# Arrays and Vectors (C# Arrays and Lists)

## Quick Run

```bash
dotnet run --project example/arrays-and-vectors-example.csproj
```

## Topics Covered

- Fixed arrays and `List<T>`
- Iteration with `for` and `foreach`
- Aggregations over collections
- Reverse traversal

## Exercise Focus

- `exercises/01.cs`: read values and print them in reverse order.
- `exercises/02.cs`: count frequency of a target value.
