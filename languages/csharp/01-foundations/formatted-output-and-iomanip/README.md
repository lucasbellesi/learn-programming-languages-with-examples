# Formatted Output and iomanip Parity (C#)

## Quick Run

```bash
dotnet run --project example/formatted-output-and-iomanip-example.csproj
```

## Topics Covered

- Alignment and width in interpolated strings
- Numeric precision formatting (`F2`, `F4`)
- Table-like output for reports
- Consistent readable output structure

## Exercise Focus

- `exercises/01.cs`: print an invoice-like aligned table.
- `exercises/02.cs`: print statistics with configurable precision.
