# Types and Input/Output (C#)

## Quick Run

```bash
dotnet run --project example/types-and-io-example.csproj
```

## Topics Covered

- `int`, `double`, `string`, `bool`
- `Console.WriteLine` output
- `Console.ReadLine` input and parsing

## Exercise Focus

- `exercises/01.cs`: read N values and compute summary.
- `exercises/02.cs`: parse product, price, quantity.
