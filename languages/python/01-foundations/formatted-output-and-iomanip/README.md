# Formatted Output and iomanip Parity (Python)

## Quick Run

```bash
python example/main.py
```

## Topics Covered

- f-string alignment (`<`, `>`, width)
- Numeric precision formatting (`:.2f`, `:.4f`)
- Fixed-width table output
- Readable report formatting

## Exercise Focus

- `exercises/01.py`: print an invoice-like aligned table.
- `exercises/02.py`: print statistics using user-selected precision.
