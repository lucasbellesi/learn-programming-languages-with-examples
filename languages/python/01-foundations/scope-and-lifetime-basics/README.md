# Scope and Lifetime Basics (Python)

## Quick Run

```bash
python example/main.py
```

## Topics Covered

- Function-local scope
- Global constants for shared configuration
- Variable lifetime inside loops and blocks
- Avoiding accidental variable reuse

## Exercise Focus

- `exercises/01.py`: classify score into grade with clean variable scope.
- `exercises/02.py`: compute running sum while keeping variable lifetime minimal.
