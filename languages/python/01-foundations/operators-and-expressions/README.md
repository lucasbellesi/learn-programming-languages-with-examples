# Operators and Expressions (Python)

## Quick Run

```bash
python example/main.py
```

## Topics Covered

- Arithmetic operators: `+ - * / // % **`
- Comparison operators: `== != < <= > >=`
- Logical operators: `and`, `or`, `not`
- Operator precedence in expressions

## Exercise Focus

- `exercises/01.py`: convert total seconds into `hours:minutes:seconds`.
- `exercises/02.py`: apply discount and tax to a purchase amount.
