# Concept Template

Use this template when creating a new concept module.

## Learning Objectives

- Define what the student should understand after studying this concept.
- Keep objectives measurable and practical.

## Explanation

Explain the concept with short sections, examples, and common pitfalls.

## Folder Layout

```text
concept-name/
  README.md
  example/
    main.cpp
  exercises/
    01.cpp
    02.cpp
```

## Checkpoint

- [ ] I understand the core idea.
- [ ] I can run and explain the example.
- [ ] I solved exercise 01.
- [ ] I solved exercise 02.
